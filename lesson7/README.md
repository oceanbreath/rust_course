#Using <command> <input>

When using modifiers: 'uppercase', 'lowercase', 'no-spaces', 'slugify' in interactive mode, use a command like:
```
no-spaces h e l l o
```

To read a csv file in CLI, use:
```
path/filename.csv
```
instead of a text line or 
```
csv path/filename.csv
```
in interactive mode.
